# parallel-ml

# Installation

## Prerequisites
- Python 3.4 +
- Pip3
- Virtualenv

## Modules
Install the following modules:
    - 'pip install tensorflow'
    - 'pip install matplotlib'
    - 'pip install pyqt5'

NOTE: Add "matplotlib.use('Qt5Agg')" or  create a ~/.matplotlib/matplotlibrc file containing 'backend: Qt5Agg'

# Run the training script
Run 'python test.py'
